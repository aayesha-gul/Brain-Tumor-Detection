# Brain-Tumor-Detection
This project predicts the presence of a brain tumor using a dataset of patient features. The system uses machine learning techniques to classify whether a patient has a tumor or not.
Features:

1 Predicts tumor presence from dataset features

2 Uses machine learning for classification

3 Works with numeric/tabular patient data

4 Provides accuracy and performance metrics

5 Simple and easy to use interface

Technologies Used:
1 Python
2 Pandas
3 NumPy
4 Scikit-learn
